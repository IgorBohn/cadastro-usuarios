
# Frontend - Cadastro de Usuários

Este é o frontend da aplicação de cadastro de usuários, desenvolvido com React, TypeScript e Material UI.

## Tecnologias

- React
- TypeScript
- Material UI
- Axios

## Instalação

1. Instale as dependências:
```bash
yarn install
# ou
npm install
```
```bash
yarn start
# ou
npm start
```
